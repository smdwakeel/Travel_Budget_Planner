# Travel Budget Planner Dashboard

---

## Project Overview

This Excel-based Travel Budget Planner helps users estimate and manage travel expenses efficiently. The project provides a structured approach to planning trips by tracking transportation, accommodation, food, and miscellaneous expenses while calculating overall savings.

---

## Tech Stack Used

* **Tool:** Microsoft Excel
* **Data Processing:** Excel Formulas
* **Analysis:** Pivot Tables & Data Summarization
* **Visualization:** Excel Charts & Reports
* **Data Source:** Travel Budget Dataset

---

## Features & Insights

* **Travel Cost Tracking:** Monitor transportation, hotel, food, and shopping expenses.
* **Budget Monitoring:** Compare planned budget with actual expenses.
* **Savings Analysis:** Automatically calculate remaining savings after trip expenses.
* **Pivot Table Reporting:** Generate summarized insights by destination and expense category.
* **Formula-Based Automation:** Uses Excel formulas for calculations and financial tracking.

---

## Dashboard Metrics

### Budget Performance

* Total Budget Allocated

### Expense Breakdown

* Transportation Cost

* Shopping & Miscellaneous Expenses

### Travel Insights

* Destination-wise Spending Analysis
* Cost Distribution by Category
